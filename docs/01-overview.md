# 概览

X-Ray 是一个用于监控 Twitter/X 的内容系统。它从观察名单与个人账号数据中拉取内容，经过 AI 分析后输出杂志风格的 Markdown 报告，并可同步到 Obsidian。

## 目标

- 收集观察名单推文与个人分析数据
- 识别高价值内容并结构化输出
- 将结果沉淀为可阅读的报告
