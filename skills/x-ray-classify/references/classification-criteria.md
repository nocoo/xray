# Classification Criteria Reference

## Interest Areas

Primary focus: **AI / LLM / Agent**

### Core Keywords

| Category | Keywords |
|----------|----------|
| **LLM Models** | GPT, Claude, Gemini, Llama, Mistral, Phi, PaLM, BERT, transformer |
| **AI Companies** | OpenAI, Anthropic, Google DeepMind, Meta AI, Hugging Face, Cohere |
| **Agent Concepts** | agent, agentic, autonomous, tool use, function calling, MCP, ReAct |
| **AI Techniques** | RAG, fine-tuning, RLHF, DPO, prompt engineering, embeddings, inference |
| **Infrastructure** | CUDA, GPU, TPU, vLLM, TensorRT, quantization, GGUF, ONNX |

### Signal Boosters (increase relevance score)

- Mentions of new model releases
- Benchmark results or comparisons
- Open source releases
- Research paper announcements (arXiv links)
- Technical tutorials or explanations
- Tool/library announcements

### Signal Reducers (decrease relevance score)

- Pure promotional content
- Job postings
- Personal opinions without technical substance
- Memes (unless technically insightful)
- Off-topic discussions

## Author Context

Consider the author's typical content:
- Known AI researchers → higher baseline relevance
- Tech company official accounts → announcements likely important
- General tech influencers → evaluate content individually

## Language Notes

Tweets may be in English or Chinese. Apply the same criteria regardless of language.
